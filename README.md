## Git Flow APK

Este repositório contém uma aplicação simples em **Node.js**, criada com o objetivo de **testar e demonstrar o uso do Git Flow** no controle de versionamento.

O projeto é utilizado para praticar conceitos como:
- Organização de branches (`main`, `develop`, `feature`, `release`, `hotfix`)
- Fluxo de trabalho colaborativo com Git
- Boas práticas de versionamento e integração de código

Não se trata de uma aplicação em produção, mas sim de um **ambiente de estudo e experimentação**.
